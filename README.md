# CloudFormation-YAML-Templates

* This Repo includes some of the widely used aws services, cloudformation YAML template.
* All the templates are very basic and easy to understandable. (less complex)

# Terraform Commands:

- Initiate File:
  ```terraform init```
- Validate File:
  ```terraform validate```
- Run File:
  ```terraform apply```
- Destroy File:
  ```terraform destroy```
